Kyooz Playlists:

  • Can be fully modified (tracks can be rearranged/deleted) within Kyooz
  • Are managed only by Kyooz
  • Do not sync with the iTunes Music Library (but they will still be backed up when a full backup is performed within iTunes)


iTunes Playlists:

  • Can only created or have songs added to them within Kyooz (available only from iOS 9.3 onwards)
  • Can only have songs rearranged or deleted using the Apple Music app or iTunes
  • Can only be deleted using the Apple Music app or iTunes
  • Sync with the iTunes library
